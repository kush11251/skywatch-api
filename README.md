## Introduction
This API provides real-time information on astronomical events.

## Getting Started
1. Clone the repository: `git clone https://github.com/user/skywatch-api.git`
2. Navigate to the project directory: `cd skywatch-api`
3. Install dependencies: `npm install`
4. Start the server: `node src/index.js`

## API Endpoints
### GET /events
Returns a list of upcoming astronomical events.
